# VALIDIKA image library

Place optimized, royalty-free photos here. Prefer WebP or AVIF exports at the exact names used by `src/lib/media.ts`.

Selection rules:
- Use recognized photo libraries such as Unsplash, Pexels, or Pixabay.
- Prioritize African and Black pharmacists, patients, and healthcare professionals, with a natural mix of women, men, and generations.
- Keep the color grade bright, clean, modern, and consistent.
- Avoid generic generated illustrations, 3D renders, artificial staging, and visibly dated stock photos.
- Export responsive sizes when needed, then keep the canonical desktop file in the folder below.

Expected files:
- `hero/validika-pharmacy-hero.webp`
- `pharmacists/pharmacist-patient-consultation.webp`
- `pharmacy/qr-code-document-verification.webp`
- `backgrounds/pharmaceutical-laboratory.webp`
- `medicines/medicine-shelves-counter.webp`

Until these local files are added, the interface uses curated remote fallbacks from recognized photo-bank search endpoints.
